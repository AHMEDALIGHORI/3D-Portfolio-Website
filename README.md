## Features

- Full-screen dark hero with Kanit typography and gradient heading text.
- Magnetic mouse-following portrait interaction.
- Scroll-responsive marquee rows with 21 motion preview GIFs.
- Character-by-character About text reveal powered by Framer Motion.
- White Services section with bold numbered service rows.
- Sticky project-card stack on desktop with responsive mobile cards.
- Lucide React icons for polished action buttons.

## Responsive Polish

- Mobile-first nav spacing and button sizing.
- Fluid hero scale for the longer `Ahmed` heading.
- Responsive marquee tiles using `clamp()` sizing.
- Service rows stack cleanly on narrow screens.
- Project cards switch from sticky desktop stacks to natural mobile cards.
- Image grids collapse to one column on phones and return to split layout on larger screens.
